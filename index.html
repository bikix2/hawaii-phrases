🎯 万能パターンを使いこなす
1. 「Can I ○○?」を使いこなせば万能!

Can I get / Can I have / Can I try / Can I pay / Can I use など、応用範囲が無限大!

2. 「○○, please.」は魔法の言葉

どんな単語の後に付けても丁寧になる最強フレーズ!

3. 「I'm ○○」で状態を伝える

I'm hungry / I'm thirsty / I'm full / I'm tired など、体調管理に必須!

4. 「It's ○○」で感想を伝える

It's hot / It's cold / It's delicious など、会話が弾みます!

5. 指差し+フレーズが最強!

迷ったら「This one, please.」で乗り切れます。

6. 番号注文が一番簡単

メニューに番号があれば、発音を気にせず確実に伝わります。

💬 コミュニケーションのコツ
1. 焦らず、ゆっくり話せばOK

ハワイの人は優しく対応してくれます。笑顔を忘れずに!

2. 分からなければ素直に聞き返す

「Excuse me?」「Sorry?」で聞き返せば大丈夫。

3. ジェスチャーと笑顔が最強の武器

言葉が完璧でなくても、笑顔とジェスチャーで伝わります!

4. メモやスマホを活用

住所、商品名などはメモで見せるのも◎

🎯 レストランでの基本マナー

1️⃣ 入店時の振る舞い
• 勝手に席に座らない - 必ずホスト/ホステスの案内を待つ
• 予約がある場合は名前と人数を伝える
• 大人数（6人以上）は予約推奨

2️⃣ 注文のタイミング
• メニューを閉じる = 注文準備完了のサイン
• ウェイターが "Ready to order?" と聞きに来る
• まだ決まっていない場合は "A few more minutes, please"

3️⃣ 食事中のコミュニケーション
• ウェイターを大声で呼ばない
• アイコンタクトや軽く手を上げる
• "Excuse me" と丁寧に声をかける

4️⃣ チップの文化
• レストラン: 15-20%が相場
• 大人数（6人以上）: 18%が自動加算される場合あり
• カウンターサービス（ファストフード）: チップ不要
• カフェ（カウンター注文）: $1-2 or 10-15%

5️⃣ 大人数グループのポイント
• 予約時に正確な人数を伝える（10人の場合は "Party of ten"）
• 会計前に別会計か一緒かを決めておく
• Gratuityが自動加算されているか確認
• 席順に注文を取られることが多い

6️⃣ アレルギー・食事制限がある場合
• "I'm allergic to ○○" = ○○アレルギーです
• "I'm vegetarian" = ベジタリアンです
• "No pork, please" = 豚肉なしで
• "Gluten-free, please" = グルテンフリーで

7️⃣ ファストフード・カフェの違い
• カウンターで注文＆支払い
• 席は自由（勝手に座ってOK）
• 番号札を渡されることが多い
• セルフサービスが基本
• チップは通常不要（カフェは少額のTip Jarあり）

📝 薬局で買える薬の種類
痛み止め・解熱剤:
• Tylenol (タイレノール) = アセトアミノフェン、解熱鎮痛剤
• Advil / Motrin (アドビル/モトリン) = イブプロフェン、痛み止め
• Aleve (アリーブ) = ナプロキセン、長時間効く痛み止め

風邪薬:
• Dayquil (デイクイル) = 昼用総合風邪薬（眠くならない）
• Nyquil (ナイクイル) = 夜用総合風邪薬（眠くなる）
• Mucinex (ミューシネックス) = 去痰薬（痰を出す）
• Robitussin (ロビタシン) = 咳止めシロップ

胃腸薬:
• Pepto-Bismol (ペプトビスモル) = 胃痛・下痢・吐き気
• Tums (タムズ) = 胃酸を中和、胸焼け
• Imodium (イモディウム) = 下痢止め
• Gas-X (ガスエックス) = ガス・膨満感

アレルギー薬:
• Benadryl (ベナドリル) = 抗ヒスタミン剤（眠くなる）
• Claritin (クラリチン) = アレルギー薬（眠くならない）
• Zyrtec (ジルテック) = アレルギー薬（24時間効果）
• Allegra (アレグラ) = アレルギー薬（眠くならない）

乗り物酔い:
• Dramamine (ドラマミン) = 酔い止め
• Bonine (ボニーン) = 酔い止め（眠くなりにくい）

その他:
• Neosporin (ネオスポリン) = 抗生物質入り軟膏（切り傷）
• Hydrocortisone cream = ヒドロコルチゾンクリーム（かゆみ止め）
• Visine (バイシン) = 目薬
• Cepacol (セパコール) = 喉の痛みのトローチ

📏 チップ・単位
💵 チップの基本ルール
ハワイ（アメリカ）ではチップは必須です！サービスに対する感謝の気持ちを示す文化です。
🍽️ レストランでのチップ
基本相場:
• 普通のサービス: 15-18%
• 良いサービス: 18-20%
• とても良いサービス: 20-25%
📝 チップの計算方法
簡単な計算:
1. 会計金額の税抜き前の金額を確認
2. 15%の場合: 金額 × 0.15
3. 20%の場合: 金額 × 0.20

例: 会計が$50の場合
• 15% = $50 × 0.15 = $7.50
• 20% = $50 × 0.20 = $10.00

もっと簡単な計算:
• 10%: 金額の1桁を右にずらす（$50 → $5）
• 20%: 10%の2倍（$5 × 2 = $10）
• 15%: 10%の1.5倍（$5 × 1.5 = $7.50）
💳 チップの渡し方
1. クレジットカードの場合:
• レシートの「Tip」欄に金額を記入
• 「Total」欄に合計を記入
• サインする

例:
Subtotal: $50.00
Tax: $2.08
-----------------
Total: $52.08
Tip: $10.00
-----------------
TOTAL: $62.08
Signature: _________

2. 現金の場合:
• テーブルに現金を置いていく
• または会計時に「Keep the change（おつりは取っておいて）」と言う
